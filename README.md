# TEMOERATURE-CONVERTER-TOOL
🌡️ Temperature Converter Tool
A simple and clean web tool to convert temperatures between Celsius, Fahrenheit, and Kelvin — built with Python and Streamlit.
🎯 About the Project
A beginner-friendly utility project built as a first-year B.Tech AI/ML student to practice:

Python arithmetic and formula implementation
Building interactive UI with Streamlit
Deploying a real working tool on the web

🛠️ Tech Stack

Language: Python 3
Web Framework: Streamlit
Deployment: Streamlit Community Cloud

⚙️ How It Works

User enters a temperature value and selects the input unit
The tool instantly converts it to all other units using standard formulas:

°F = (°C × 9/5) + 32
K = °C + 273.15


Result is displayed cleanly on screen

🚀 How to Run Locally
bashpip install streamlit
streamlit run app.py
